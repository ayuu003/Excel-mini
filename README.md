# 📊 Project Title

## 📝 Overview
This project focuses on analyzing a dataset related to **[brief description of the project, e.g., cinema tickets and receipts across Lombardy provinces]**. The objective is to extract meaningful insights, calculate key performance indicators (KPIs), and present the results in a clear and structured manner.

## 🎯 Objectives
- Analyze the provided dataset to identify trends and patterns.
- Sort records based on **takings** in descending order.
- Calculate total **tickets sold** and **revenue generated**.
- Derive key performance indicators such as **Average Ticket Price**.
- Present findings using data visualization techniques.

## 📂 Dataset
- **Format:** Excel (.xlsx)
- **Key Columns:**
  - Province
  - Number of Tickets
  - Revenue (Takings)
  - Date (if applicable)

## 🛠️ Tools & Technologies
- **Microsoft Excel** – Data cleaning and analysis
- **Tableau / Power BI** – Data visualization (if used)
- **Git & GitHub** – Version control and collaboration

## 📈 Methodology
1. **Data Cleaning**
   - Removed duplicates and handled missing values.
   - Ensured consistency in data formats.

2. **Data Sorting**
   - Sorted 100% of records by **takings** in descending order.

3. **Calculations**
   - **Total Tickets Sold:** `SUM(Tickets)`
   - **Total Revenue:** `SUM(Revenue)`
   - **Average Ticket Price (KPI):**
     ```
     Average Ticket Price = Total Revenue ÷ Total Tickets
     ```

4. **Visualization**
   - Created charts and dashboards to represent key insights.
